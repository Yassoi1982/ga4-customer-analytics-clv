# Segment Definitions (Plain English)

## RFM Segments
- Champions: recent + frequent + high spend
- Loyal: recent + repeat buyers
- Potential Loyalist: fairly recent but not frequent yet
- At Risk: used to buy often but not recent
- Hibernating: very old activity + low frequency
